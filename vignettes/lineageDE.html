1 - Introduction

This vignette will describe the workflow of the R package lineageDE, showing two different methods to infer differentially expressed genes from reconstructed cell lineages. After a step of cell ordering reconstruction performed with Slingshot (see (K. Street et al. 2017)) we try to find genes wit significantly different behaviour between two cell lineages.

This vignette will use the data from (Fletcher et al. 2017) of Olfactory Epithelium cells from mouses. We restricted the data to only two cell lineages for simplicity purpose.

Cell lineages used in the study

1.1 - Preprocessing of the data for our analysis

Before searching for differentially expressed genes between lineages, we have to preprocess the data and run slingshot on it. We use data from (Fletcher et al. 2017), which is already filtered and normalized with Scone . Scone tries different normalization techniques, batch or technical effects removal methods and provides tools to rank this different normalizations.

We load this data from the package lineageDE which contains also the different functions that we use in this study. We also load the package slingshot (K. Street et al. 2017) which allow us to perform the lineage reconstruction

library(lineageDE)
library(slingshot)
# For nice color
library(RColorBrewer)
data("OE_counts")

We first transform the data into logcounts (a usual transformation to have gene counts on the same scale: WHY!!!???) and we perform a pca as the dimension reduction technique:

pca <- prcomp(t(log1p(counts)))
plot3d(pca$x[,1:3], col = clusters)

We apply slingshot to infer the lineages curves:

slrun <- slingshot(pca$x[,1:3], clusters, start.clus = '1', end.clus = '4')
# color for the clusters
clust_col = brewer.pal(length(unique(clusters)),"Set3")[as.numeric(clusters)] 
plot3d(pca$x[,1:3], col = clust_col)
plot3d.SlingshotDataSet(slrun, type = "curves", add = T)
# generate 2D captures of the 3d plots for the rmarkdown
par(cex.main = 1)
plot(pca$x[,1:2], col = clust_col, pch=16, asp = 1)
for(c in curves(slrun)){ 
  lines(c$s[c$tag,c(1,2)], lwd = 1)
}
title("View of the first two Pcs of the data with principal curves")

Then, we store the slingshot outputs for our future DE analysis:

times <- pseudotime(slrun)
weights <- curveWeights(slrun)
w1 <- weights[,1]/(weights[,1] + weights[,2])
w2 <- weights[,2]/(weights[,1] + weights[,2])
w <- cbind(w1,w2)
